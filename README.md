# Projeto Pokenext

## Descrição

Um aplicativo que lista e exibe detalhes dos Pokémon usando a PokéAPI.

## Tecnologias Usadas

- React
- Next.js
- Tailwind CSS

## Funcionalidades

- Listar Pokémons
- Exibir detalhes de cada Pokémon

## API Utilizada

- PokéAPI
